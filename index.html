Rules of the Game:

  1. This game can be played in two modes: Player vs Computer or Player vs Player!
  2. Each player must first fill the username which can contain only small or big letters on the range of [3,16] letters.
  3. Player must select the mode of the game that they want to play against each other!
    • Best of 3: There will be exatly 3 winning games(Draws doesn't count as winning game), "Best of 5" and "Best of 9" are similiar.
  4. After each winning game the game counter on top-left will increment(This means for draws will not).
  5. After on or other player wins, it's name will be highlight with a red for losing and green for winning and the game will be reset after 2 seconds automatically.
  6. The Reset Buton can be used at any time the player wants to reset the game.
  7. Choose below the mode of the game: Enjoy 😊
